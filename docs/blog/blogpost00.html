Fig.2 alfred loci, circa 19whatever



The loci method is a method of memorization which involves mentally visualizing a location along with anything that you are trying to memorize. As you try to memorize more items, you should imagine more of this mental location, so that each item has a unique placement within it. By doing so, you are utilizing a part of your brain called the hippocampus, which is reponsible for remembering the layout of physical locations. Unless you suffer from the condition called Aphantasia, this technique can be used to not only memorize large quantities of data, but it can also be used to help visualize the order in which items are arranged.
